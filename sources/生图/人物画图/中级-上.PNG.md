## Summary

这张中级上篇海报把人物图目标从“像人”推进到“像真实的人”。它的核心公式是“身份 + 状态 + 行为 -> 真实的人物瞬间”，并提出三层细化：生理层、状态层、行为层。重点不是增加装饰词，而是让人物拥有身份、当下状态和正在发生的行为。

## Source Digest

图像标题为“让人物‘真实’”，副标题是从“像人”到“像真实的人”。顶部目标写明要提升语义密度，让画面更真实、更有生活感；核心表达是“身份 + 状态 + 行为”共同构成真实的人物瞬间。

第一部分是“三层细化（让模型脑补）”。生理层决定人物基础外观，描述要点包括脸型、五官、发型、肤色、体型、穿搭风格等；状态层决定人物情绪与状态，描述要点包括情绪、气质、疲惫程度、专注/放松/紧张等；行为层被标为最关键，作用是让人物“活”起来，描述要点包括动作、姿态、视线方向和正在做的事。三层叠加后，人物会从静态形象变成具体场景中的人。

第二部分通过示例对比说明语义密度不足的问题。不足示例是“一个漂亮女生，站在街上”，问题包括信息太少、没有状态、没有动作、画面空洞。完整示例把人物改为“刚下班的年轻女性”，补充略带疲惫但自然精致的脸、地铁车厢、扶手、手机发消息、车厢灯光偏冷、环境略显拥挤、写实风格等。右侧标注的优点是有身份、有状态、有动作、有环境，因此更真实。

第三部分总结中级关键点：身份和状态比“颜值词”更重要；动作让画面活起来；环境提供真实感，场景、时间和环境细节会影响画面的可信度。

第四部分把“真实感”拆成五类可写设定：身份设定、状态设定、动作设定、场景设定、光线/时间设定。示例中身份可以是大学生、设计师、刚下班的职场人或旅行中的女生；状态可以是略显疲惫、心情放松、有些烦躁、专注认真；动作可以是低头看手机、整理头发、扶着扶手、手里拿着咖啡；场景可以是地铁车厢、雨后的街道、咖啡店角落、办公室内；光线和时间可以是傍晚路灯刚亮、清晨阳光柔和、夜晚霓虹灯光、室内暖色灯光。

第五部分给出中级完整模板，把身份、外表、动作+细节、具体场景+时间、环境补充细节、具体光线、风格和焦点位置串起来。图中附有使用前自检清单：是否有身份、状态、动作、场景、光线、风格和焦点。
