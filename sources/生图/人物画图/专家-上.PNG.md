## Summary

这张信息图是人物生图专家篇上半部分，主题是“叙事与氛围”，目标是把“好看的人物”升级成“有故事的瞬间”。它强调画面不是静止物，而是一个正在发生的瞬间；提示词要把前因后果、情绪、环境、光线、构图和色彩组织在一起。

## Source Digest

该图的核心方法是用“故事瞬间”替代静态人物描述。示例从“一个女孩站在街上”升级为“女孩刚下完雨，站在街角路灯下，手里拿着透明雨伞，望向远处，像是在等一个人”。这类描述通过刚刚发生、正在发生、似乎发生等时间状态，让画面带有情绪和叙事张力。

情绪与氛围部分用表格把情绪、环境、光线和色调关联起来。孤独/思念可以对应夜晚街道或空旷场景、逆光或侧后光、冷色调；温暖/治愈可以对应咖啡馆或午后阳光、柔光或侧光、暖色调；紧张/不安可以对应昏暗室内或封闭空间、顶光或局部光源、低饱和和高对比。这说明人物生图的情绪不是单靠表情词实现，而是由环境、光线和色彩共同传递。

构图与视觉引导部分强调构图决定观众视线如何移动。图中列出三分法构图、引导线构图、框架式构图、留白构图：三分法让主体落在交点或三分线上；引导线利用道路、栏杆和光影；框架式构图用门窗或树枝形成自然框架；留白用于突出孤独和空间感。

色彩与氛围部分把冷色调、暖色调、低饱和、高对比分别映射到不同情绪：冷色调对应孤独、冷静、科技感、距离感；暖色调对应温暖、治愈、幸福、亲近感；低饱和对应文艺、怀旧、情绪内敛；高对比对应戏剧性、冲突感、强烈情绪。图中提醒“色彩服务于情绪，不要为了好看而用色，要为了表达而用色”。

专家级提示词模板被组织为：人物设定 + 状态与情绪 + 故事瞬间前因后果 + 场景环境 + 时间天气 + 光线氛围 + 构图视角 + 色彩风格 + 画面重点。这个模板比普通提示词更强调变量之间的因果关系和一致性。
