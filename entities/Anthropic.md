# Anthropic

## What It Is

Anthropic 是 Claude 模型的创建方。当前 wiki 中，它作为系统提示语案例、Claude Code skills 实践和 AI 研发自动化论述的来源出现。

## Role In This Wiki

`案例.xmind` 收集了 Claude/Anthropic 风格的系统提示语片段，用于学习如何表达助手身份、时间、知识边界、能力边界、输出习惯、多模态安全和失败处理。新增 human sources 让 Anthropic 也成为 Claude Code skills 运营和 AI 研发自动化治理的证据来源。

## Key Facts

- Claude 系统提示语案例展示了如何把模型身份、能力、限制和回答风格写成长期行为契约。
- 图像相关提示语案例强调人脸识别限制、隐私边界和图像内指令处理。
- 当前 source 还提到 Claude 3 系列，但这些时间敏感信息未在本次 ingest 中联网核验。
- Anthropic blog source 披露 Claude Code skills 的内部分类、gotchas、progressive disclosure、hooks、memory 和 marketplace 路径。
- Anthropic Institute source 把 AI 研发自动化拆为工程执行、实验优化、人类审查/验证瓶颈和治理窗口。
