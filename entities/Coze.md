# Coze

## What It Is

Coze 在本 wiki 中被整理为低代码 Bot / Agent 平台案例，围绕 Bot、知识库、卡片、工具、记忆、工作流和 API 能力组织。

## Role In This Wiki

它用于比较低代码 Agent 平台和代码优先 Agent Runtime 的边界：快速组合 RAG、工作流、多 Agent 分发和渠道交互，但全局状态和深度定制能力受限。

## Key Facts

- Bot 可以采用单 Agent、多 Agent、工作流三种编排模式。
- 工作流支持循环和并发，但不如代码灵活。
- 长期记忆会总结对话并在后续回复时召回。
- 知识库支持 RAG，但 source 记录 PDF 还原精度不足。
- 当前产品状态未联网核验。
