# AI Wiki Hotwords

本文件维护本仓库播客转写的默认热词。`ai-wiki-cook-podcast` 会在转写前默认读取这里的术语，并通过 Whisper `initial_prompt` 做 spelling bias。

规则：

- 一行一个术语，或用逗号/顿号分隔多个术语。
- 只放高频、稳定、容易被 ASR 误写的词。
- 单集特有的人名、公司、产品、节目名由 Agent 从 Apple metadata 和单集描述临时补充，不必全部写进这里。
- 这不是强制词典，只是转写提示；仍需在最终 note 里说明转写模型和限制。
