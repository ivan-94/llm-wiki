# 单变量迭代（视觉投放）

## Definition

单变量迭代是商品图批量生产中的优化方法：固定商品与模板不变，每次只改变一个变量（场景/光线/背景色/文案），再用投放数据选择最优版本。它是 [[concepts/电商商品图生成|电商商品图生成]] 把生图从“技巧”升级为“运营系统”的关键。

## Why It Matters

批量生产真正的竞争力是“可复制、可迭代的稳定产出”，而不是偶然做出一张爆图。一次改多个变量会让投放结果难以归因；单变量迭代降低归因难度，让设计变化和投放结果之间形成反馈闭环。

## Mental Model

模板化 + 单变量 + 数据闭环：

1. 模板库固定不变部分：版式、光线、场景、色调模板。
2. 每次只改一个变量，生成原图对照组 + 版本 A/B/C/D。
3. 用曝光量、点击率、转化率、成交金额对比选择最优版本。
4. 把胜出版本沉淀回素材库，持续迭代。

标准化流程：明确需求 → 素材准备 → 构思方案 → AI 生成 → 筛选优化 → 应用投放。

## Key Claims

- 高级阶段重点是“批量生产：稳定输出，持续优化”。
- 单变量迭代每次只改一个变量，便于归因。
- 用曝光量、点击率、转化率、成交金额等数据选择最优版本。
- 稳定产出靠固定商品一致性、固定风格体系、模板化 prompt、素材库沉淀。

## Examples

- 背景色单变量：保留同一瓶身，生成原图与版本 A/B/C/D，选点击率/转化率最高的。

## Common Confusions

- 一次改多个变量——结果无法归因，难以复盘。
- 把“批量”理解成多出几张——其实是建立可交接、可复盘的流程资产。
