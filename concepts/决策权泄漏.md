# 决策权泄漏

## Definition

决策权泄漏是指本该由人类、产品、领域规则、安全边界或架构约束决定的事项，被 Agent 在实现过程中静默决定。

## Why It Matters

泄漏的决策会被写进代码，表面上像实现细节，实际上改变了产品语义、权限模型、错误处理、数据状态或系统边界。

## Mental Model

决策权泄漏是 Agent 把“未定义”误当成“可以自由补全”。

## Key Claims

- 权限例外、状态含义、错误码、校验层级和批量入口规则都可能是高层决策。
- 暴露所有决策不可行，因为决策具有分形特征；需要管理决策分辨率。
- L1 语义层和 L2 边界层应强治理，普通 L4 实现细节可以放权。
