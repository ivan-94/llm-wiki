# 电商详情页模块

## Definition

电商详情页模块是把电商视觉内容拆成承担不同功能的图片模块，并按用户浏览逻辑组合，完成“吸引用户 → 传递信息 → 促成转化”。它是 [[concepts/电商商品图生成|电商商品图生成]] 从“单图生成”升级到“模块化信息架构”的方法。

## Why It Matters

电商页面不是一堆漂亮图的堆叠，而是一条转化漏斗。每张图都该有明确任务；模块顺序错了，再好看也无法引导用户走完决策路径。

## Mental Model

一图一功能，按转化漏斗排列：

1. 商品主图：第一印象——主体清晰、干净、突出。
2. 卖点图：解决痛点——图文结合，卖点明确。
3. 场景图：增强代入——真实自然、贴近生活。
4. 使用图/模特图：建立信任——动作自然，突出效果。
5. 参数图：打消顾虑——信息准确、排版清晰。

组合技巧：顺序清晰、重点突出、风格统一、合理留白。

## Key Claims

- 电商页面是多模块组合，各模块共同完成吸引、传递、转化。
- 五大核心模块：主图、卖点图、场景图、使用图/模特图、参数图。
- 推荐顺序即转化漏斗：主图 → 卖点 → 场景 → 使用/模特 → 参数。
- 生成单图也要围绕详情页的信息节奏来规划功能。

## Examples

- 主图先抓注意，卖点图传递利益，场景图增强代入，使用图建立信任，参数图打消顾虑。

## Common Confusions

- 把详情页当“多张漂亮图”——其实是按购买决策顺序组织信息。
- 每张图都想说全部信息——应一图一功能。
