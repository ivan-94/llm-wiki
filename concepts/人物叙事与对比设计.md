# 人物叙事与对比设计

## Definition

人物叙事与对比设计是人物生图的专家级能力：用“故事瞬间”替代静态人物描述，并通过对比关系（光线、明暗、动静、虚实）提升画面表现力。它对应 [[concepts/人物生图提示词|人物生图提示词]] 四层模型的第四层——叙事一致性。

## Why It Matters

到了专家级，画面要从“好看”升级为“有故事感、专业级视觉表达”。叙事让人物画面有时间感和情绪张力，对比让画面有层次和冲突感；二者都要求人物、环境、光线、色彩指向同一个情绪目标。

## Mental Model

不是描述画面，而是导演一个画面：

- 故事瞬间：用“刚刚发生 / 正在发生 / 似乎要发生”赋予时间感。
- 情绪由组合传递：情绪 = 环境 + 光线 + 色调，而非单靠表情词。
- 对比设计：每张图至少设计一组对比（光线/明暗/动静/虚实）。
- 细节克制：从环境、人物、天气时间、氛围线索里只选 1-3 个最强化氛围的元素。

## Key Claims

- 专家级目标是让画面有故事感和情绪张力，而不只是好看人物。
- 情绪需要由环境、光线、色调和示例描述共同传递。
- 色彩是情绪语言：冷色调=孤独/距离，暖色调=温暖/亲近，低饱和=文艺内敛，高对比=戏剧冲突。
- 高级感来自对比关系；细节只需选 1-3 个，不堆砌。
- 专业工作流：灵感构思 → 参考收集 → 关键信息设定 → 提示词撰写 → 生成微调 → 优化输出。

## Examples

- 故事瞬间：女孩刚下完雨，站在街角路灯下，手里拿透明雨伞，望向远处，像在等一个人。
- 完整公式：人物设定 + 状态情绪 + 动作细节 + 场景叙事 + 光线氛围 + 镜头构图 + 风格质感。

## Common Confusions

- 把“叙事”理解成写一段长故事——其实是让所有变量指向同一情绪。
- 为“高级”堆满细节——细节过多反而杂乱，要克制到 1-3 个。

## My Understanding

专家级人物图是在导演一个有前因后果的瞬间，并用一组对比把它讲得更有张力。
