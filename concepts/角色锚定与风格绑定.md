# 角色锚定与风格绑定

## Definition

角色锚定与风格绑定是人物生图保持跨场景一致性的两个稳定模块：角色锚定指先建立并反复复用一套核心人物特征；风格绑定指选定一种明确风格并长期贯穿。它是 [[concepts/人物生图提示词|人物生图提示词]] 中对应“锁定不可变对象”的部分。

## Why It Matters

人物图最大的隐患是漂移——同一角色跨图脸不一致、风格忽变。角色锚定让“同一个人”有据可依，风格绑定避免多风格混用导致的画面割裂，二者共同把人物变成“可被反复拍摄的角色”。

## Mental Model

把人物当成一个可复用的角色档案：

- 角色锚定：写一段固定的核心特征描述，在不同场景里原样复用。
- 风格绑定：从一组明确风格里选一种，长期使用，不混搭。

人物图的“锁定层”，对应商品图的 [[concepts/约束编辑（商品生图）|约束编辑]]。

## Key Claims

- 角色锚定要求先设定并反复复用核心特征，有助于跨场景一致。
- 风格绑定应选一种风格贯穿始终，避免多风格混用。
- 复杂参考图、反复迭代和混乱风格会提高漂移风险。
- 人物图最好一次出图；多次迭代可能让背景劣化、出现涂抹感。

## Examples

- 角色设定：22 岁亚洲女性、脸型偏圆、五官柔和、大眼睛双眼皮、鼻梁小巧、唇形饱满、长直发黑棕色、气质温柔安静。
- 风格绑定：电影感写实摄影 / 日系清新写实 / 胶片风格人像 / 韩系自然光人像 / 暗调情绪人像，选其一长期用。

## Common Confusions

- 以为写“同一个人”就能保持一致——需要可复用的具体特征描述。
- 为追求新鲜频繁换风格——会破坏作品集的统一气质。
